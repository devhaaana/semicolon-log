## 갑자기 JS, CSS 파일이 적용되지 않는다?

JS, CSS 파일을 수정하고 바로 반영되지 않는 경우가 있다.
브라우저 캐시에 수정하기 전의 JS, CSS 파일이 남아있기 때문이다.

### 새로고침

크롬의 경우 `강력 새로고침`을 하면 된다.

- Windows OS

```
Ctrl + Shift + R
```

- Mac OS

```
Command + Shift + R
```

### 캐시 삭제

브라우저 `캐시 및 쿠키를 삭제`하면 된다.

- Windows OS

```
Ctrl + Shift + Delete
```

- Mac OS

```
Command + Shift + Delete
```
